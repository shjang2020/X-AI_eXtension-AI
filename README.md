# 🤖 X-AI (eXtension-AI)

다양한 도메인의 딥러닝 논문 리뷰 및 구현을 목표로 하는 X-AI 학회에서 진행한 내용용입니다.

## 📋 개요

### 1. 컴퓨터 비전
- YOLO (객체 검출)
- U-Net (이미지 세그멘테이션)
- SSD (Single Shot Detector)
- RetinaNet (객체 검출)
- MobileNet (경량화 모델)

### 2. 생성 모델
- StyleGAN (이미지 생성)
- DCGAN (이미지 생성)
- Seq2Seq (시퀀스 생성)
- Audio (음성 생성)

### 3. 그래프 신경망
- GNN (Graph Neural Network)
- 그래프 임베딩
- 노드 분류
- 링크 예측

## 🛠️ 사용 기술
- Python
- PyTorch
- TensorFlow
- CUDA
- OpenCV

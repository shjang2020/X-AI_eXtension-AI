# 🔄 GNN (Graph Neural Network)

그래프 데이터 처리를 위한 논문 리뷰 프로젝트입니다.

## 📋 프로젝트 개요

### 1. 논문 리뷰
- 그래프 신경망에 관한 최신 연구 동향
- GCN, 노드 분류, 그래프 분류, 링크 예측에 대한 논문 분석

### 2. 주요 내용
- 그래프 데이터 처리 기술
- 노드 임베딩 알고리즘
- 그래프 구조 학습 방법론

### 3. 응용 분야
- 소셜 네트워크
- 추천 시스템
- 분자 구조 분석

## 📄 논문 요약
- 그래프 데이터의 특징 추출 및 처리 방법에 대한 심층 분석
- 최신 그래프 신경망 모델을 활용한 노드 및 그래프 분류 기법
- 링크 예측을 위한 데이터셋 및 평가 방법

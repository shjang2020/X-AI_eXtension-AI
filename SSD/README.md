# 🎯 SSD (Single Shot Detector)

단일 단계 객체 검출을 위한 SSD 모델 구현 프로젝트입니다.

## 📋 프로젝트 개요

### 1. 구현 모델
- 기본 SSD
  - 객체 검출
  - VOC 데이터셋 학습
  - 다중 스케일 특징

### 2. 주요 기능
- 단일 단계 검출
- 다중 스케일 처리
- 실시간 객체 검출

### 3. 응용 분야
- 객체 검출
- 자율주행
- 보안 시스템

- OpenCV
- CUDA
